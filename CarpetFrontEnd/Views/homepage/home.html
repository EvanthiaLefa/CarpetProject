Carpet Project

A Contact CRUD Application!

The TechPro Academy Carpet Project is a CRUD application that allows you to manage your contact database with ease. You can organize and manage a thorough list of contacts and keep track of important information like name, email address, phone number, city and state using the four major functionality create, view, update, and delete. Designed to help us refine our technical skill set up to this point. The following are the main technologies we're utilizing to build this application: Java, SPRING BOOT, JavaScript, HTML, BOOTSTRAP, and MYSQL!
