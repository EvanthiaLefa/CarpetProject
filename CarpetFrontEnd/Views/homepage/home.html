Carpet Project

A Contact CRUD Application!

The TechPro Academy Carpet Project is a CRUD application that allows you to manage your contact database with ease. With four main functionalities - create, view, update, and delete - you can organize and maintain a comprehensive list of contacts and keep track of important details such as name, email address, phone number, and organization. Designed in order to practice our technical skill set until now. Main technologies we using is JAVA : [ SPRING, JDBC DRIVER ] JS BOOTSTRAP MYSQL !
